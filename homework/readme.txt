leaf.vcxproj
    这是使用应用程序向导生成的 VC++ 项目的主项目文件，其中包含生成该文件的 Visual C++ 的版本信息，
    以及有关使用应用程序向导选择的平台、配置和项目功能的信息。

leaf.vcxproj.filters
    这是使用“应用程序向导”生成的 VC++ 项目筛选器文件。它包含有关项目文件与筛选器之间的关联信息。
    在 IDE 中，通过这种关联，在特定节点下以分组形式显示具有相似扩展名的文件。例如，“.cpp”文件与“源文件”筛选器关联。

工程测试与使用方法与说明

1.使用vs2012打开解压后的project文件夹下的leaf文件（工程文件：文件类型为Microsoft Visual Studio Solution）；

2.编译生成可执行文件；

3.运行程序，输入测试数据以test.txt文档存在于project文件夹中，故请在控制台输入该文件路径；

4.程序从test.txt读取测试数据存放于CountProfit类的string input中。传入String generateSummary(String input)后得到输出，并打印显示；

5.完毕；

注：修改测试输入，直接修改test.txt内容即可。
